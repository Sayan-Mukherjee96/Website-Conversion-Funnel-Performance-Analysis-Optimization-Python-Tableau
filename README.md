# Website-Conversion-Funnel-Performance-Analysis-Optimization
Analyzing and optimizing website conversion funnel performance to improve user journey and increase conversions using Python and Tableau.


# 📌 Overview
This project uses Python and Tableau to analyze website conversion funnel performance, identify drop-offs, and generate actionable insights to improve conversions and business outcomes.

# 🎯 Problem Statement
Despite high traffic from marketing channels, the website shows low conversion efficiency due to user drop-offs at key stages of the funnel, especially on mobile, impacting overall revenue performance.

# ⭐ Dataset
Multiple CSV files located in  / data/ folder(Order, Session, Refund, order items, products, pageviews) 
Summery table created from ingested data and used for analysis

# 📊 Tools & Technologies
Python (Numpy, Pandas, Matploylib, Seaborn, Plotly)
Tableau(Interactive Visualizations)

# Exploratory Data Analysis(EDA)
1. Desktop drives higher traffic and revenue, while mobile shows growth potential.
2. Gsearch dominates performance; other channels underperform.
3. Revenue and sessions peaked in 2014, then declined slightly in 2015.
4. Improved basket size indicates better cross-selling and monetization.

# 🔍 Research Questions & Key Findings
1. Device Performance:
◦ Desktop generates significantly higher sessions and revenue than mobile → indicates stronger user engagement and conversion efficiency on desktop

2. Conversion Funnel Drop-offs:
◦ Highest drop-off occurs at checkout stage (Mobile ~71.7%, Desktop ~54.2%) → highlights checkout as the primary bottleneck in the funnel

3. Mobile vs Desktop Conversion:
◦ Mobile shows consistently lower conversion rates compared to desktop → suggests gaps in mobile user experience and optimization

4. Channel Performance:
◦ Gsearch drives the highest traffic and revenue but shows relatively lower conversion efficiency → indicates mismatch between traffic acquisition and conversion quality

5. Revenue Trend:
◦ Average revenue increased significantly until 2014, followed by a slight decline in 2015 → reflects improved monetization initially but challenges in sustaining growth

6. Traffic Trend:
◦ Sessions grew steadily until 2014 but dropped sharply in 2015 → indicates potential issues in traffic acquisition or marketing strategy

7. Campaign Effectiveness:
◦ Purchase contribution from campaigns increased nearly 24x in 2014 → shows significant improvement in campaign performance and targeting

8. Revenue Efficiency:
◦ Revenue per session increased by ~21% → indicates improved conversion efficiency and monetization
